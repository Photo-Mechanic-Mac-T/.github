## ЁЯУД Overview

<a href="#photoMechanic">Photo Mechanic</a> is the professional photographer's culling and workflow tool тАФ famous in the photojournalism and commercial photography industry for providing the fastest RAW photo viewing and rating experience available on Mac, enabling photographers to process large shoots in a fraction of the time required by Lightroom or other combined editing and culling tools.

The <a href="#photoMechanic">instant preview rendering</a> is Photo Mechanic's defining capability тАФ instead of generating previews from RAW data (which is computationally intensive and creates the sluggish browsing that Lightroom users experience with large RAW files), Photo Mechanic reads the embedded JPEG preview that cameras write into every RAW file alongside the RAW data. These embedded previews display instantly, enabling photographers to flip through hundreds of frames per minute rather than waiting for each RAW file to render. For sports photographers, wedding photographers, and photojournalists who shoot thousands of frames per session, this speed difference is the difference between a culling session that takes hours and one that takes minutes. The <a href="#photoMechanic">star rating and color tag system</a> applies ratings and tags through single keyboard shortcuts тАФ pressing a number key applies a star rating to the current image and immediately advances to the next, making the culling process a rapid keyboard-driven review where the photographer keeps eyes on the images rather than reaching for menus or trackpad gestures.

The <a href="#photoMechanic">IPTC metadata editing</a> provides the industry-standard metadata fields that photojournalists, stock photographers, and commercial photographers need тАФ caption, headline, credit, copyright, location, keywords, and the complete IPTC standard тАФ applied to single images or batch-applied to selections of images at once. Metadata embedded directly into image files travels with the images through publication workflows, editorial systems, and stock submission. The <a href="#photoMechanic">batch rename capability</a> renames entire shoots according to configurable naming conventions тАФ date, time, sequence number, job name, and any combination of variables тАФ producing consistently named file libraries that professional archive management requires.

<a href="#photoMechanic">Memory card ingest workflow</a> copies images from camera memory cards with configurable destination folder creation, simultaneous backup to a second location, automated renaming during ingest, and metadata application at import тАФ the complete card-to-archive workflow in a single operation. <a href="#photoMechanic">GPS mapping integration</a> displays captured GPS coordinates on a map for images that contain location metadata тАФ useful for documentary photographers, travel photographers, and any workflow where geographic context accompanies the images.

<a href="#photoMechanic">Contact sheet view</a> displays grids of thumbnail images for visual review across an entire shoot тАФ enabling rapid visual scanning to identify the selects within a shoot before detailed individual image review. <a href="#photoMechanic">Integration with Lightroom and Photoshop</a> sends selected images directly to Lightroom for editing or Photoshop for retouching тАФ Photo Mechanic handles the culling and metadata, editing applications handle the image processing, in a combined workflow that uses each tool for what it does best.

Photo Mechanic's speed advantage comes from a fundamental architectural decision тАФ reading embedded JPEG previews rather than rendering from RAW data. This isn't a hardware or optimization difference; it's a workflow philosophy difference. Photo Mechanic's developers recognized that for culling purposes, the embedded preview is sufficient to make accept/reject decisions, and that RAW rendering should happen only for the selects that will actually be edited. This insight, implemented early in Photo Mechanic's development, remains its defining competitive advantage over Lightroom and other tools that try to do both culling and editing in the same application.

## ЁЯТм FAQ

<details>
<summary>Why is Photo Mechanic faster than Lightroom for viewing RAW files?</summary>

Photo Mechanic reads the embedded JPEG preview that cameras write into every RAW file, which displays instantly. Lightroom renders previews from the RAW data, which is computationally intensive and slower. For culling purposes, the embedded preview is sufficient to make selection decisions.

</details>

<details>
<summary>Does Photo Mechanic edit photos?</summary>

Photo Mechanic focuses on culling, rating, metadata, and workflow management. It does not perform pixel-level photo editing. Selected images export to Lightroom, Photoshop, or other editing applications for post-processing.

</details>

<details>
<summary>What is IPTC metadata?</summary>

IPTC is the industry-standard metadata schema for photographic images тАФ including fields for caption, headline, credit, copyright, location, and keywords. Photo Mechanic provides full IPTC metadata editing that embeds into image files for use in editorial and stock workflows.

</details>

<details>
<summary>Can Photo Mechanic ingest from memory cards?</summary>

Yes. Photo Mechanic's ingest feature copies images from memory cards with configurable destination folders, simultaneous backup, automated renaming, and metadata application during import.

</details>

<details>
<summary>Does Photo Mechanic work with Lightroom?</summary>

Yes. Selected images send directly from Photo Mechanic to Lightroom for editing. Many photographers use Photo Mechanic for culling and Lightroom for editing in a combined workflow.

</details>

<details>
<summary>What is Photo Mechanic Plus?</summary>

Photo Mechanic Plus adds a catalog database (DAM) component to Photo Mechanic's core speed for library management across the full photo archive, combining culling speed with catalog organization.

</details>

<details>
<summary>Can I apply ratings with keyboard shortcuts?</summary>

Yes. Number keys apply star ratings and advance to the next image immediately тАФ the keyboard-driven culling workflow that makes Photo Mechanic fast in practice.

</details>

<details>
<summary>Does Photo Mechanic support batch metadata editing?</summary>

Yes. IPTC metadata applies to single images or batches of selected images simultaneously.

</details>

<details>
<summary>Does Photo Mechanic show GPS coordinates on a map?</summary>

Yes. Images with GPS metadata display location on a map within Photo Mechanic.

</details>
